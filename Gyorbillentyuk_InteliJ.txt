

---

### IntelliJ IDEA gyorsbillentyűk (Windows/Linux)

| Gyorsbillentyű       | Mit csinál?                                              |
| -------------------- | -------------------------------------------------------- |
| **Shift + F6**       | Átnevezés (Refactor → Rename)                            |
| **Ctrl + Space**     | Alapvető kódkiegészítés (Basic Code Completion)          |
| **Ctrl + Shift + P** | Aktuális kifejezés/típus megjelenítése (Expression Type) |
| **Ctrl + P**         | Paraméter javaslatok megjelenítése metódushíváskor       |
